## 前程无忧爬虫
### 项目
```
・ pip安装scrapy：pip install scrapy
・ 创建scrapy项目：scrapy startproject mySpider
```
在当前目录下输入命令，将在mySpider/spider目录下创建一个名为itcast的爬虫
```
・ scrapy genspider job51 "51job.com"
```
启动爬虫项目
```
・ scrapy crawl job51
```
